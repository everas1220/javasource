입출력(input/output)


-기본출력 : 키보드
-기본출력 : 화면

-System.out 화면
-System.in 키보드
  -new 

-입력장치(키보드, 파일,객체)에서 읽어오기
 1) text파일 :문자스트림
 2) text +모든 파일 : 바이트 스트림

- 스트림(Stream) : 서로 연결된 두대산의 연결통로
 -스트림은 양방향 사용불가 
 - 입력,출력은 따로 처리

- File :자바에서 파일(텍스트,이미지,동영상)처리를 담당하는 객체
 -1) 윈도우에서 사용한느 경로 구분자: ; (세미콜론)
 -2) 유닉스(리눅스) : :(콜론)

 - 윈도우에서 사용하는 이름 구분자 : \
 - 유닉스(리눅스) : /
